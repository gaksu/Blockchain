About the Application

Data Share aims at creating a system where the users can share files securely using a blockchain. Each connected user stores the complete blockchain.

Why a Blockchain?

  • Decentralized environment
  • Supply chain data transparency
  • Secure data ledgers due to the sophisticated encryption mecahnisms
  • Data stored on the blockchain can't be deleted or tampered with
